# CatMusic 音乐播放器项目总结

## 一、项目概述

CatMusic 是一款基于 Android 平台开发的音乐播放器应用，旨在提供流畅、美观的音乐播放体验。该应用支持音乐播放、暂停、上一首、下一首、循环播放、随机播放等基本功能，同时还实现了歌词同步显示、专辑封面旋转动画、进度条同步等增强功能。

## 二、技术栈

| 技术/框架 | 用途 |
|---------|------|
| Android | 应用开发平台 |
| Java | 主要开发语言 |
| OkHttp | 网络请求 |
| Glide | 图片加载和缓存 |
| MediaPlayer | 音频播放核心组件 |

| Binder | Activity 与 Service 通信 |
| Custom View | 自定义歌词显示控件 |
| XML | 布局和资源文件 |

## 三、项目架构

CatMusic 采用了典型的 Android 应用架构，主要分为以下几个层次：

1. **UI 层**：负责用户界面的显示和交互，包括 PlayerActivity、SongListActivity 等
2. **业务逻辑层**：负责处理业务逻辑，包括 LyricBiz 等
3. **服务层**：负责后台音乐播放，包括 MusicService
4. **数据层**：负责数据的获取和解析，包括 LyricParser 等
5. **工具层**：提供通用的工具类和方法

## 四、核心功能实现

### 1. 音乐播放功能

- **实现方式**：使用 Android 系统提供的 MediaPlayer 组件进行音频播放
- **核心功能**：
  - 播放/暂停音乐
  - 上一首/下一首切换
  - 循环播放（列表循环、单曲循环）
  - 随机播放
  - 进度条同步和拖动跳转

### 2. 歌词同步功能

- **实现方式**：自定义 LyricView 控件，通过定时任务同步歌词显示
- **核心功能**：
  - 歌词解析（支持多种格式）
  - 歌词实时同步
  - 歌词高亮显示
  - 歌词滚动效果

### 3. 专辑封面功能

- **实现方式**：使用 ImageView 显示专辑封面，结合动画实现旋转效果
- **核心功能**：
  - 专辑封面加载和缓存
  - 封面旋转动画
  - 封面背景和阴影效果

### 4. 服务通信功能

- **实现方式**：使用 ServiceConnection 和 Binder 进行 Activity 与 Service 之间的通信
- **核心功能**：
  - 服务绑定和解除绑定
  - 播放状态同步
  - 播放控制命令传递

## 五、优化点

### 1. 歌词解析优化

- **问题**：歌词格式不规范，时间戳位置不固定
- **解决方案**：修改 LyricParser 类，支持多种歌词格式，包括：
  - 标准格式：[mm:ss.xx]歌词内容
  - 非标准格式：歌词内容[mm:ss.xx]
- **优化效果**：能够正确解析各种格式的歌词，提高了应用的兼容性

### 2. 歌词同步优化

- **问题**：歌词没有随着歌曲播放实时更新
- **解决方案**：
  - 在 onPlay 方法中启动歌词同步
  - 在 onPlaybackPause 和 onStop 方法中停止歌词同步
  - 使用 Handler + Runnable 实现定时更新
- **优化效果**：歌词能够随着歌曲播放实时更新，提供了良好的同步体验

### 3. 专辑封面优化

- **问题**：专辑封面旋转动画不流畅，背景设计简单
- **解决方案**：
  - 调整旋转动画持续时间，从 10 秒增加到 15 秒
  - 优化背景设计，使用 layer-list 实现双层效果
  - 添加阴影效果，增强立体感
- **优化效果**：封面旋转更加流畅，背景设计更加立体，视觉效果更好

### 4. 歌词显示优化

- **问题**：歌词中包含换行符和时间戳，显示不整洁
- **解决方案**：
  - 在歌词解析阶段移除换行符和转义换行符
  - 在歌词显示阶段再次处理转义换行符
- **优化效果**：歌词显示整洁，没有多余的换行符和时间戳

## 六、遇到的问题和解决方案

### 1. 歌词解析格式问题

- **问题**：服务器返回的歌词格式不规范，时间戳出现在歌词内容后面
- **解决方案**：修改 LyricParser 类，添加对非标准格式歌词的支持
- **代码位置**：`LyricParser.java:36-140`

### 2. 歌词同步问题

- **问题**：歌词没有随着歌曲播放实时更新
- **解决方案**：在播放状态变化时，正确启动和停止歌词同步
- **代码位置**：`PlayerActivity.java:835-941`

### 3. 专辑封面显示问题

- **问题**：专辑封面旋转动画不流畅，背景设计简单
- **解决方案**：优化旋转动画和背景设计
- **代码位置**：
  - `PlayerActivity.java:237-250`
  - `album_art_background.xml:1-36`
  - `activity_player.xml:23-35`

### 4. 歌词中包含换行符问题

- **问题**：歌词中包含换行符和转义换行符，显示不整洁
- **解决方案**：在歌词解析和显示阶段处理换行符
- **代码位置**：
  - `LyricParser.java:38`
  - `LyricView.java:179`
